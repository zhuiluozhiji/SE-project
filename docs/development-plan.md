# 开发计划
ai 乱写的 不管他
## 第 1 周

- 前端：页面骨架、路由、Mock 数据。
- 后端：FastAPI 骨架、统一响应、Swagger、基础接口占位。
- 数据库：核心表结构、种子数据。
- 爬虫：确定数据源，完成字段抽取方案。
- 测试文书：测试计划、接口字段表、演示脚本初稿。

## 第 2 周

- 前端联调活动列表、详情、日历。
- 后端实现活动、推荐、日程、课程导入核心接口。
- 爬虫导入一批真实或半真实活动数据。
- 测试文书开始记录接口联调问题。

## 第 3 周

- 完成冲突检测、ICS 导出、后台管理。
- 进行第一轮系统测试和 Bug 修复。

## 第 4 周

- 打磨演示数据、测试报告、用户手册和答辩材料。
